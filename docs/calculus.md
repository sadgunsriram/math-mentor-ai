# Calculus Rules

## Derivative Rules

### Power Rule

d/dx (x^n) = n * x^(n-1)

Example:

d/dx (x^3) = 3x^2

---

### Constant Rule

d/dx (c) = 0

Example:

d/dx (5) = 0

---

### Sum Rule

d/dx (f + g) = f' + g'

Example:

d/dx (x^2 + x)

= 2x + 1

---

### Product Rule

(fg)' = f'g + fg'

Example:

d/dx (x * sin x)

---

### Chain Rule

If y = f(g(x))

dy/dx = f'(g(x)) * g'(x)

Example:

d/dx (sin(x^2))

---

## Basic Derivatives

d/dx (sin x) = cos x

d/dx (cos x) = -sin x

d/dx (tan x) = sec^2 x

---

## Important Limits

lim (x → 0) (sin x)/x = 1

lim (x → 0) (1 - cos x)/x = 0
