# Linear Algebra Basics

## Matrix

A matrix is a rectangular array of numbers arranged in rows and columns.

Example:

|1 2|
|3 4|

---

## Determinant of 2x2 Matrix

For matrix:

|a b|
|c d|

Determinant:

det(A) = ad - bc

Example:

|1 2|
|3 4|

det = (1 × 4) - (2 × 3)

= 4 - 6

= -2

---

## Eigenvalues

Eigenvalues satisfy:

det(A - λI) = 0

Example:

Matrix:

|2 0|
|0 3|

Eigenvalues:

λ₁ = 2  
λ₂ = 3

---

## Identity Matrix

|1 0|
|0 1|

Multiplying any matrix by identity matrix gives same matrix.

---

## Matrix Multiplication Rule

A(m × n) × B(n × p) = C(m × p)
